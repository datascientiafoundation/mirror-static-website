This talk will present our going work, started more than 6 years ago, whose goal is to enable data integration among data collected in different cities, about different but related topics, in different languages, thus enabling cross-language data management. The first part will present the theoretical foundations. The second part will discuss the concrete steps towards making this project concrete, namely the creation of a no-profit organization whose goal will be to enable interoperability among the different sites.<\/div>\n
